# Licensing and provenance

Record provenance before packaging and update it after material edits.

```json
{
  "generated": true,
  "createdAt": "2026-07-15T18:00:00.000Z",
  "generator": {
    "type": "ai-image-generation",
    "provider": "user-supplied-or-agent-recorded",
    "model": "user-supplied-or-agent-recorded"
  },
  "humanDirection": "Approved style specification and anchor",
  "referenceAssets": [],
  "modifications": ["cropped", "centered", "resized", "converted to webp"],
  "source": "Project design workflow",
  "ownership": "License status: user confirmation required",
  "license": "License status: user confirmation required"
}
```

## Rules

- Record the real generator, provider, model, creation time, human direction, references, and modifications when known.
- Use `License status: user confirmation required` when rights cannot be verified.
- Never convert unknown status into “safe”, “owned”, “royalty free”, MIT, or public domain.
- Never copy Apple, Samsung, or other proprietary vendor artwork or ask a model to reproduce it closely.
- Treat a visual reference as directional input only when its use is authorized.
- Do not use a logo, mascot, character, or trademark without explicit authority.
- Do not upload private source code, credentials, customer data, or an entire repository to an image generator.
- Keep generated outputs local until human review.

Generated artwork can still carry contractual, trademark, likeness, or model-provider restrictions. The skill records evidence; it does not provide legal clearance.
